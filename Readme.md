# Insight & Reflection Section

**An LLM-powered tool for analyzing quarterly 10-Q reports from public companies using LangChain and Streamlit. 
During development, I experimented with various LLM models including `google/flan-t5-base` and `google/flan-t5-large`. The smaller model offered faster responses but with slightly reduced detail, while the base model provided more detailed answers.

Switching from FAISS to Chroma for vector storage solved deployment challenges related to system dependencies, improving app reliability in cloud environments.

---

## 💡 Features
- Extract key financial, operational, and risk insights from SEC 10-Q reports
- Built with LangChain for document processing and Hugging Face Transformers for natural language understanding
- Intuitive Streamlit UI for uploading PDFs and querying reports interactively
- Utilizes Chroma vector store for efficient document retrieval
- Choose between FLAN-T5-Small (faster, less detailed) and FLAN-T5-Base (slower, more detailed) directly in the app sidebar.

---

## 🧰 Tech Stack
- **LangChain:** Document loaders, splitters, embeddings, and retrieval chains  
- **Chroma:** Vector similarity search 
- **Hugging Face Transformers** – Q&A via `google/flan-t5-base` and `google/flan-t5-large`
- **Streamlit:** Web app interface

## Evaluation Metrics

| Model | Conciseness | Numeric Accuracy | Context Understanding |
|-------|------------|-----------------|---------------------|
| Base  | 2          | 5               | 4                   |
| Large | 4          | 5               | 3                   |

### Key Observations from Model Outputs

- **Base Model:** Includes extra context, document references, and is very verbose.  
- **Large Model:** Extracts the core insight (like cash flow or revenue) and is more readable.  
- **Both models:** Agree on straightforward answers such as company name or revenue.  
- **Tradeoff Highlighted:** This illustrates the balance between detail (Base) and conciseness/readability (Large).

## 📖 Related Work & Acknowledgements

This project focuses on **practical financial document analysis** using publicly available SEC 10-Q filings. No research papers were directly reproduced; instead, the project leverages open-source tools and frameworks:

- **LLM Models:** `google/flan-t5-base` and `google/flan-t5-large` from [Hugging Face Transformers](https://huggingface.co/models) for text generation and summarization.  
- **Vector Databases:** Chroma for storing and retrieving document embeddings efficiently.  
- **Document Processing Frameworks:** LangChain for document loading, splitting, embeddings, and retrieval chains.  
- **UI & Deployment:** Streamlit for an interactive web interface to query reports.  

We also acknowledge the **SEC EDGAR database** as the source of the 10-Q reports used in this project. These reports are publicly available and provided the core dataset for testing and evaluation.  

This project responsibly uses these open-source tools and adheres to their licensing terms.
